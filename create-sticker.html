Cara Menambahkan ke WhatsApp

1

Buat & Unduh Pack

Isi semua persyaratan grid stiker di atas, lalu tekan tombol Unduh Pack (.wastickers).

2

Gunakan Aplikasi Tambahan

Pastikan HP Anda sudah menginstal aplikasi pembaca `.wastickers` ini. Jika belum punya, unduh secara gratis di bawah ini:

Play Store App Store
3

Buka / Share File

Cari file `.wastickers` di folder Download. Jika saat diklik malah terbuka seperti folder, tekan tahan file tersebut > pilih Bagikan (Share) > pilih Sticker Maker, atau buka aplikasinya lalu pilih Import.
